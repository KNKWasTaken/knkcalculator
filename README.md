## KNK Calculator

This is a calculator created in python by KNKWasTaken

It supports:-
1. ~Addition~
2. ~Subtraction~
3. ~Multiplication~
4. ~Division~
5. ~Average (Mean)~
6. ~Percentage~

**Note:-** This is a template that can be used for different programs

**Note:-** This uses a function (`exit()`) that is only applicable in IDLE (python)
